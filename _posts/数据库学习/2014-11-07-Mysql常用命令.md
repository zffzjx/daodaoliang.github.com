### 给予Root用户最高权限

> 
mysql 1449 ： The user specified as a definer ('root'@'%') does not exist 解决方法

```sql
mysql> grant all privileges on *.* to root@"%" identified by ".";
 Query OK, 0 rows affected (0.00 sec)

mysql> flush privileges;
 Query OK, 0 rows affected (0.00 sec)

```
